Core Capabilities

一块屏幕，看清宏观全貌

从地缘地图到量化雷达，从逻辑推演到资讯流，所有决策要素在一块大屏上协同运转。

可视化宏观地图

地缘冲突、军事部署、自贸区、供应链航线一图呈现，点击标记点即弹出区域实时情报。

多维量化雷达

政治、货币、产业、供应链等八维 0-1 评分，雷达图实时联动，一眼对比不同资产强弱。

核心逻辑推演

阶段化流程卡片（阶段 1/2/3）拆解当前标的研究逻辑，把"为什么买"讲清楚。

动态事件信息流

底部滚动财经资讯、大宗商品价格、机构评级与相关标的池，关键信号不漏接。

相关标的池

按量化评分排序的关联标的，评分环直观呈现强弱，辅助快速构建观察清单。

开放 API 接入

各维度均预留接口位与 Mock 数据源，真实数据通过外部 API 动态注入，组件零改动。
